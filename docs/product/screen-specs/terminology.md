# 제품 용어 기준

작성 기준일: 2026-07-10

| 표준 용어 | 정의 | 코드·DB 대응 | 사용하지 않을 표현 |
| --- | --- | --- | --- |
| 파트너사 | 싸트너십과 계약하고 하나 이상의 제휴처를 운영하는 회사 또는 사업자 | `partner_companies`, `companyId` | 협력사, 업체(회사 의미일 때) |
| 제휴처 | 구성원에게 노출되는 서비스·브랜드 단위 | `partners`, `partnerId` | 브랜드, 서비스, 파트너(노출 단위 의미일 때) |
| 지점 | 한 제휴처의 혜택이 적용되는 실제 장소 | branch·location 계열 필드 | 매장(온라인 제휴까지 포괄할 때) |
| 혜택 | 구성원에게 제공되는 할인·증정·링크 이용 조건 | `benefits`, `conditions`, `benefit_action_*` | 프로모션(홈 광고 의미와 혼동될 때) |
| 내 인증 | 로그인한 구성원이 제시하는 SSAFY 인증 카드와 QR | `/certification` | 인증센터, 구성원 카드 |
| 내 알림 | 현재 계정이 수신한 알림함 | member/admin/partner notification inbox | 알림 운영, 발송 관리 |
| 발송 관리 | 관리자가 대상·채널·내용을 정해 알림을 보내고 결과를 확인하는 기능 | `/admin/push` | 내 알림 |

코드 식별자는 호환성을 위해 즉시 바꾸지 않는다. 사용자 문구, 화면 제목, 내비게이션에는 표준 용어를 사용하고 저장소·DB 용어는 매핑 문서에서만 병기한다.
