## 제거 안전장치

제거 판정 화면은 같은 wave에서 메뉴와 링크를 제거하고 기존 URL은 canonical 경로로 리다이렉트한다. 데이터·테이블 삭제는 Production 한 안정화 기간이 지난 뒤 사용량과 참조를 다시 확인하는 별도 migration으로 수행한다.
