# 제휴처 쿠폰 생성 및 사용방법

제휴처 담당자용 안내 문서입니다. 쿠폰은 회원이 제휴처 상세에서 다운로드하고, 개인 쿠폰함에서 확인한 뒤 사용할 수 있습니다. 운영 권한에 따라 관리자 제휴처 상세 또는 파트너 포털에서 쿠폰을 관리합니다.

## 1. 쿠폰 생성·수정 방법

1. 관리자 계정으로 로그인합니다.
2. `제휴처 관리`에서 쿠폰을 운영할 제휴처의 상세 페이지로 이동합니다.
3. `제휴처 쿠폰` 영역에서 쿠폰을 생성합니다.
4. 회원에게 보여줄 쿠폰 이름, 혜택, 다운로드 기간, 사용 기간과 발급 한도를 입력합니다.
5. 현장에서 인증 카드와 함께 확인할 쿠폰이라면 사용 방식에서 `현장 확인`을 선택하고, 숫자 4자리 PIN을 설정합니다.
6. 저장된 쿠폰은 목록에서 `수정`, `복제`, `삭제`할 수 있습니다.

파트너 포털 권한이 있는 담당자는 `파트너 포털 → 회사 → 서비스 상세 → 쿠폰 관리`에서 신규 쿠폰과 파트너 코드 목록을 등록할 수 있습니다. 쿠폰의 수정·복제·삭제는 관리자 제휴처 상세의 `제휴처 쿠폰` 영역에서 진행합니다.

복제한 쿠폰은 새 `초안`으로 만들어집니다. 기존 기간·혜택·발급 한도 등은 복사되므로 다른 혜택명과 필요한 내용만 수정한 뒤 활성화하면 됩니다. 파트너 코드 목록과 쿠폰 사용 이력은 복사하지 않습니다. 이미 발급·사용 이력이 있는 쿠폰은 삭제할 수 없으며, 종료 상태로 변경해 주세요.

## 2. 회원에게 쿠폰 노출 및 다운로드

1. 쿠폰 상태를 `활성`으로 변경하고 다운로드 기간을 확인합니다.
2. 회원은 제휴처 상세 페이지에서 쿠폰 내용을 확인합니다.
3. 회원이 `쿠폰 다운로드`를 누르면 쿠폰이 개인 쿠폰함에 저장됩니다.
4. 회원은 `/coupons`에서 보유 쿠폰의 사용 기간, 남은 사용 횟수, 코드 또는 외부 링크를 확인할 수 있습니다.

일·주·월 발급 한도와 회원별 일·주·월 한도는 입력한 값에 따라 적용됩니다. 값을 입력하지 않은 한도는 무제한으로 운영됩니다.

## 3. 현장 확인형 쿠폰 사용방법

### 📌 사용 방법 (매장 직원 기준)

1. 회원이 실시간 `싸트너십 인증 카드`와 사용할 쿠폰 화면을 함께 제시합니다.
2. 직원은 인증 카드의 회원 정보와 쿠폰의 혜택·사용 기간을 확인합니다.
3. 직원이 쿠폰 화면의 현장 확인 영역에서 제휴처 전용 숫자 4자리 PIN을 입력합니다.
4. 인증 성공 화면을 확인한 뒤 혜택을 적용합니다.

PIN은 직원이 직접 입력합니다. 회원에게 PIN을 물어보거나 PIN을 쿠폰 설명·메시지·캡처 화면에 적어 전달하지 마세요.

## 4. 코드 제시형·외부 링크형 쿠폰

- `코드 제시` 쿠폰은 회원이 개인 쿠폰함에서 발급된 쿠폰 코드를 보여주면 확인 후 사용 처리합니다.
- 파트너 코드형 쿠폰은 파트너가 제공한 코드 목록을 한 줄에 하나씩 등록합니다. 코드는 회원별로 발급되며, 남은 코드가 없으면 새로 다운로드할 수 없습니다.
- `외부 링크` 쿠폰은 회원이 쿠폰함의 링크를 통해 제휴처가 지정한 외부 서비스에서 사용합니다.

## 📌 유의사항

- 현장 확인 PIN은 숫자 정확히 4자리입니다. 선행 0도 포함할 수 있습니다. 예: `0427`
- 인증은 실시간 화면을 기준으로 확인합니다. 캡처 이미지나 만료된 화면은 인정하지 마세요.
- 쿠폰의 다운로드 기간, 사용 기간, 전체 발급 한도, 회원별 주기 한도를 확인한 뒤 운영해 주세요.
- PIN을 변경하면 이후 발급되는 쿠폰부터 새 PIN이 적용됩니다. 이미 발급된 쿠폰에는 발급 당시의 확인 정보가 적용될 수 있습니다.
- 쿠폰 코드형의 코드 목록은 개인정보나 다른 서비스의 비밀번호를 포함하지 않는 운영용 코드만 등록합니다.
- 쿠폰 사용 오류가 반복되면 쿠폰을 임의로 삭제하지 말고 관리자에게 쿠폰명, 회원 화면, 발생 시각을 전달해 주세요. PIN 자체는 전달하지 않습니다.
