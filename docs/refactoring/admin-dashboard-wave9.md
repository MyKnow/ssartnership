# Admin Dashboard Wave 9

## 목표
- 회원 관리 유입 추이 차트의 세로 밀도를 줄이고, 시계열 해상도를 운영자가 직접 바꿔 볼 수 있게 한다.
- 각 시점의 증감 인원을 hover/click으로 바로 읽을 수 있게 만든다.

## 반영 내용
- 회원 유입 추이 차트 컨테이너 높이를 범위와 무관한 고정 높이로 더 낮춰 과도한 빈 공간을 줄였다.
- 차트 입력을 월별 사전 집계값 대신 `created_at` 원본 배열 기반으로 재구성했다.
- 다음 시계열 탭을 추가했다.
  - 일별: 최근 7일
  - 주별: 최근 1개월
  - 월별: 최근 1년
  - 연별: 전체
- 각 지점 hover/click 시 선택 구간, 변화량, 누적 인원을 상단 요약 영역에서 보여주도록 변경했다.
- 회원 관리 페이지는 새 차트 API에 맞춰 생성일 목록만 전달하도록 정리했다.

## 제외 범위
- 회원 관리 메트릭 카드의 집계 기준 변경은 이번 wave에 포함하지 않았다.
- 차트 데이터를 서버 RPC로 미리 집계하는 최적화는 아직 필요하지 않아 보류했다.
